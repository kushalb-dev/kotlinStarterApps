# BitsAndPizzas

This app was built for the purpose of polishing my viewModel concepts as well as learning
about chips and radio buttons. I am now confident with the concept and implementation of viewModels
and can handle state of the app more effectively.

The app is very simple. The user can select either of the pizzas mentioned and any number of extras
mentioned. A toast is displayed for everytime the order status changes to a non-empty order. Please
read the comments in the code for more details of the implementation.
